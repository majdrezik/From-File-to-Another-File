# From-File-to-Another-File
Given a file with names of students and 3 grades, the program prints each name with the higher grade into the same file.

Before:
Moshe 100 80 90
Dana 56 89 78
Maya 88 87 91
Adam 90 74 81

After:
Name: Moshe grade: 100
Name: Dana grade: 89
Name: Maya grade: 91
Name: Adam grade: 90
